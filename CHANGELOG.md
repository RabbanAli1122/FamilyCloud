# Changelog

All notable changes to the FamilyCloud project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-01

### Added
- Complete Ubuntu Server 24.04 LTS installation guide
- Nextcloud installation via Snap
- External storage drive setup and configuration
- Cloudflare Tunnel setup for external access
- Cloudflare domain purchase and configuration guide
- Static IP configuration for Ubuntu and routers
- Automatic backup setup for Android phones
- Automatic backup setup for iPhone
- Desktop client setup for Windows
- Desktop client setup for macOS
- Comprehensive troubleshooting section
- FAQ section covering common questions
- Detailed hardware requirements
- Network bottleneck discussion
- Free domain alternatives comparison (DuckDNS, EU.org, InfinityFree)
- SMART drive health checking instructions
- Multi-user account creation
- Storage quota management
- Contacts and calendar backup instructions

### Documentation
- README.md with complete step-by-step guide
- LICENSE file (MIT)
- CONTRIBUTING.md with contribution guidelines
- CHANGELOG.md for version tracking
- .gitignore for repository cleanliness

### Known Issues
- Snap confinement requires manual removable-media connection
- Some routers may have different static IP configuration methods
- Drive health monitoring requires manual checking
